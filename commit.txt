feat: add Quick-Setup API for OpenClaw self-configuration

New feature: Template-based database quick setup for AI agents

API endpoints:
- GET /api/v1/databases/templates - List available templates
- POST /api/v1/databases/quick-setup - Create database from template

Templates:
- ai-memory: AI Agent cross-conversation memory storage
  - user_preferences, conversation_history, knowledge_base tables
- workflow-state: Workflow state management
  - workflow_instances, workflow_events tables
- knowledge-base: Enterprise knowledge base with vector search
  - documents, tags tables

This enables OpenClaw to automatically configure oc-db9 as its memory layer
by sending a simple API request with template and name parameters.
