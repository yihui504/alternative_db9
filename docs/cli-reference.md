# OpenClaw-db9 CLI 使用文档

OpenClaw-db9 提供了一个功能完整的命令行工具 (oc-db9)，用于管理数据库、执行 SQL、生成类型定义等操作。

## 安装

### 从源码构建

```bash
# 克隆项目
git clone https://github.com/openclaw-db9/oc-db9.git
cd oc-db9

# 构建
go build -o oc-db9 ./cmd/oc-db9

# Windows
go build -o oc-db9.exe ./cmd/oc-db9
```

### 验证安装

```bash
./oc-db9 --version
./oc-db9 --help
```

---

## 全局配置

### 环境变量

| 变量 | 描述 | 默认值 |
|------|------|--------|
| `OC_DB9_API_URL` | API 服务地址 | `http://localhost:8080` |
| `OC_DB9_API_KEY` | API 密钥（未来版本） | - |

### 配置文件

创建 `~/.oc-db9.yaml` 或 `.oc-db9.yaml`：

```yaml
api_url: http://localhost:8080
output_dir: ./generated
default_lang: typescript
```

## 数据库管理 (db)

### 创建数据库

```bash
oc-db9 db create --name <数据库名> [--description <描述>]
```

**参数**

| 参数 | 简写 | 必填 | 描述 |
|------|------|------|------|
| `--name` | `-n` | 是 | 数据库名称 |
| `--description` | `-d` | 否 | 数据库描述 |

**示例**

```bash
# 创建简单数据库
oc-db9 db create --name myapp

# 创建带描述的数据库
oc-db9 db create --name myapp --description "My application database"

# 使用简写
oc-db9 db create -n myapp -d "Application DB"
```

**输出**

```
Database created successfully!
ID: 746d6a2a-b3c7-4145-a376-d1a2be69b573
Name: myapp
Connection: postgres://postgres:postgres@localhost:5432/oc_746d6a2a
```

---

### 列出数据库

```bash
oc-db9 db list [--format <格式>]
```

**参数**

| 参数 | 简写 | 描述 |
|------|------|------|
| `--format` | `-f` | 输出格式：table, json, yaml（默认 table） |

**示例**

```bash
# 表格格式
oc-db9 db list

# JSON 格式
oc-db9 db list --format json

# YAML 格式
oc-db9 db list -f yaml
```

**输出**

```

e4e6a925-79a2-43cf-ab0e-6537df208778   testdb    2026-03-21 00:10:00
```

### 删除数据库

```bash
oc-db9 db delete --id <数据库ID> [--force]
```

**参数**

| 参数 | 简写 | 描述 |
|------|------|------|
| `--id` | `-i` | 数据库 ID |
| `--force` | `-f` | 跳过确认提示 |

**示例**

```bash
# 带确认
oc-db9 db delete --id 746d6a2a-b3c7-4145-a376-d1a2be69b573

# 强制删除
oc-db9 db delete -i 746d6a2a-b3c7-4145-a376-d1a2be69b573 --force
```

### 执行 SQL

```bash
oc-db9 db sql --id <数据库ID> --query <SQL语句>
oc-db9 db sql --id <数据库ID> --file <SQL文件>
```

**参数**

| 参数 | 简写 | 描述 |
|------|------|------|
| `--id` | `-i` | 数据库 ID |
| `--query` | `-q` | SQL 语句 |
| `--file` | `-f` | SQL 文件路径 |
| `--format` | `-o` | 输出格式：table, json（默认 table） |

**示例**

```bash
# 执行单条 SQL
oc-db9 db sql --id 746d6a2a --query "SELECT * FROM users"

# 使用文件
oc-db9 db sql --id 746d6a2a --file ./migrations/001_create_users.sql

# JSON 输出
oc-db9 db sql -i 746d6a2a -q "SELECT * FROM users" -o json

# 创建表
oc-db9 db sql -i 746d6a2a -q "CREATE TABLE users (id SERIAL PRIMARY KEY, name TEXT, email TEXT)"

# 插入数据
oc-db9 db sql -i 746d6a2a -q "INSERT INTO users (name, email) VALUES ('Alice', 'alice@example.com')"
```

**输出**

```
Query executed successfully.
Results:
+----+-------+-------------------+
| id | name  | email             |
+----+-------+-------------------+
|  1 | Alice | alice@example.com |
+----+-------+-------------------+
1 row returned
```

## 代码生成 (gen)

### 生成类型定义

```bash
oc-db9 gen types --id <数据库ID> --lang <语言> [--output <目录>]
```

**参数**

| 参数 | 简写 | 描述 |
|------|------|------|
| `--id` | `-i` | 数据库 ID |
| `--lang` | `-l` | 目标语言：typescript, python, go |
| `--output` | `-o` | 输出目录（默认当前目录） |
| `--table` | `-t` | 仅生成指定表 |

---

### TypeScript 类型

```bash
oc-db9 gen types --id 746d6a2a --lang typescript
oc-db9 gen types -i 746d6a2a -l typescript -o ./src/types
```

**输出** (`types.ts`)

```typescript
// Auto-generated by oc-db9

export interface Users {
  id: number;
  name: string;
  email?: string;
  created_at?: Date;
}

export interface Posts {
  id: number;
  title?: string;
  content?: string;
  user_id?: number;
  published?: boolean;
}
```

---

### Python 类型

```bash
oc-db9 gen types --id 746d6a2a --lang python
oc-db9 gen types -i 746d6a2a -l python -o ./models
```

**输出** (`models.py`)

```python
# Auto-generated by oc-db9

from dataclasses import dataclass
from typing import Optional
from datetime import datetime

@dataclass
class Users:
    id: int
    name: str
    email: Optional[str]
    created_at: Optional[datetime]

@dataclass
class Posts:
    id: int
    title: Optional[str]
    content: Optional[str]
    user_id: Optional[int]
    published: Optional[bool]
```

---

### Go 类型

```bash
oc-db9 gen types --id 746d6a2a --lang go
oc-db9 gen types -i 746d6a2a -l go -o ./models
```

**输出** (`models.go`)


## 使用示例

### 完整工作流

```bash
# 1. 创建数据库
oc-db9 db create -n myapp -d "My Application"

# 2. 获取数据库 ID
DB_ID=$(oc-db9 db list -f json | jq -r '.[0].id')

# 3. 创建表
oc-db9 db sql -i $DB_ID -q "CREATE TABLE users (id SERIAL PRIMARY KEY, name TEXT NOT NULL, email TEXT)"

# 4. 插入数据
oc-db9 db sql -i $DB_ID -q "INSERT INTO users (name, email) VALUES ('Alice', 'alice@example.com')"

# 5. 生成类型定义
oc-db9 gen types -i $DB_ID -l typescript -o ./src/types

# 6. 创建分支
oc-db9 branch create -n dev-branch -s $DB_ID

# 7. 创建备份
oc-db9 backup create -d $DB_ID -n "initial-backup"
```

### CI/CD 集成

```yaml
# .github/workflows/db-migrate.yml
name: Database Migration

on:
  push:
    branches: [main]

jobs:
  migrate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup oc-db9
        run: |
          curl -sL https://github.com/openclaw-db9/oc-db9/releases/latest/download/oc-db9-linux-amd64 -o oc-db9
          chmod +x oc-db9
          
      - name: Run migrations
        env:
          OC_DB9_API_URL: ${{ secrets.OC_DB9_API_URL }}
        run: |
          DB_ID=${{ secrets.DATABASE_ID }}
          ./oc-db9 db sql -i $DB_ID -f ./migrations/schema.sql
          
      - name: Generate types
        run: |
          ./oc-db9 gen types -i ${{ secrets.DATABASE_ID }} -l typescript -o ./src/types
```

## 故障排除

### 连接失败

```bash
# 检查 API 服务状态
curl http://localhost:8080/health

# 检查配置
oc-db9 config view

# 设置正确的 API URL
oc-db9 config set api_url http://your-server:8080
```

### 权限问题

```bash
# Linux/macOS
chmod +x oc-db9

# Windows - 以管理员身份运行 PowerShell
Set-ExecutionPolicy RemoteSigned
```

### 调试模式

```bash
# 启用详细输出
oc-db9 --verbose db list
oc-db9 -v db sql -i $DB_ID -q "SELECT 1"
```
